◆ IsLittleEndian()

abstract bool TNet.IO.EndianBitConverter.IsLittleEndian ( )
pure virtual

Indicates the byte order ("endianess") in which data is converted using this class.

Different computer architectures store data using different byte orders. "Big-endian" means the most significant byte is on the left end of a word. "Little-endian" means the most significant byte is on the right end of a word.

Returns
true if this converter is little-endian, false otherwise.

Implemented in TNet.IO.BigEndianBitConverter, and TNet.IO.LittleEndianBitConverter.
